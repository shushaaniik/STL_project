# STL Project

## Word occupancy

**Counts occupancy of each word in given string.**

### Usage:

```
cd word_occupancy
cmake -S . -B ./build
cd build
make
../word_occupancy
```

## Inpix-Postfix

**Converts infix to postfix, and postfix to infix.**

### Usage:

```
cd inf_post
cmake -S . -B ./build
cd build
make
../inf_post
```
